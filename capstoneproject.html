Snow/Ice Melting System

Project Goal

Design and build a scaled model of a Snow and Ice Melting System which is primarily powered using a sustainable energy source for campus pavements in Ontario.

Project Overview

The team and I built a 2ft x 2ft concrete slab with PEX tubing embedded in the concrete. A small heater was used to heat water in a reservoir and a submersible pump was used to push the water through the system. The temperature sensors and snow sensors provided feedback to the NodeMCU which controlled the pump and heater using relays. The system was powered using a solar panel and a lithium-ion battery.

My Reponsibilities

Control System Lead

  • Built a control system prototype using a NodeMCU, 8-channel relay, temperature sensors, and a snow sensor that could remotely monitor and control all the components of the system through a mobile app.
  • Supported the mechanical team create a high-level CAD model of the system to show how all the parts are connected
  • Created mechanical drafts of the piping layout within the concrete
  • Performed headloss calculations for the water flowing through the pipes to determine optimal pump size
