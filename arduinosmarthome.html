Arduino - Smart Home

Project Goal

Build a smart home which can monitor and display the electricity usuage and the respective cost in real time.

Project Overview

The team and I built a model of a house with LEDs to represent home appliances. An Arduino was used to calculate the total electricity usage and the respective cost by checking which appliance (LED) was ON. The usage and cost were displayed in real-time on an LCD screen for the user. The electricity cost was based on Ontario's Hydro rates and it rates changed 3 times per day to reflect real-world scenario. Additional features of the home included smart bedroom lights which were controlled using an ultrasonic sensor and an IR sensor was used to allow the user to control any of the appliances with a remote.
